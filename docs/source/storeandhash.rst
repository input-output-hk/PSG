StoreAndHash Service - Documentation
####################################

The StoreAndHash service provides an API for posting data to `AWS S3 <https://aws.amazon.com/s3/>`_ or `IPFS <https://ipfs.io/>`_ storages, getting the hash of the data and URL for further download.

This section describes the service API and how to use it.
